### Video Downloader Web Interface

### Requirements
- Docker
- Docker Compose
- PostgreSQL Database (for login and user management)

### Install
- ```bash
  git clone --recursive https://github.com/al-pi314/videodownloader.git
  ```
- ```bash
  cd videodownloader
  ```

### How to Run Locally
- ```bash 
  docker-compose up -d --build
  ```
